## What makes the editor different

- **Drag handle on every block** — grab the handle to reorder anything, including tables and images.
- **Collapsible headings** — fold long sections away while drafting.
- **Tab completion** — let the AI finish your thought based on what you've already written.
- **Live presence** — see every collaborator's cursor and selection in real time.
- **Markdown shortcuts** — `# `, `- `, `> `, ```` ``` ```` all work the way you'd expect.

## Common shortcuts

| Action | Shortcut |
| --- | --- |
| Open the slash menu | `/` |
| Bold | `Cmd/Ctrl + B` |
| Italic | `Cmd/Ctrl + I` |
| Underline | `Cmd/Ctrl + U` |
| Insert link | `Cmd/Ctrl + K` |
| Undo / redo | `Cmd/Ctrl + Z` / `Cmd/Ctrl + Shift + Z` |

More shortcuts (AI tools, list toggles, headings) are listed in the in-app shortcut sheet — open it from the question-mark menu.
