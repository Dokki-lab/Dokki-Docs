Copilot is the AI panel for asking questions, creating content, and returning to earlier conversations without leaving the workspace.

## Start with Dokki Agent

Open the AI panel to continue your most relevant conversation or start a new one with the built-in **Dokki Agent**. You do not need to install a specialized agent before asking for everyday help.

## Give Copilot the right context

- Open Copilot from the document, table, file, or Artifact you are working on.
- Mention or attach any additional workspace resources it should use.
- State the expected result, such as a summary, decision, edit, table update, or new Artifact.
- Say whether Copilot may change the resource or should only answer in chat.

## Find and understand conversations

Use conversation filters to narrow the list. Message timestamps help you understand when a response was created, and the panel keeps your place when you move between conversations.

## Continue longer work

Some requests use tools or take longer than a simple answer. Keep the conversation open to follow progress, or return later from the conversation list. If a task should stop, use the available stop control and wait for the stopped state before starting a conflicting request.

## Access and safety

Copilot works with the access of the person or agent using it. It cannot use a private workspace resource merely because its name was mentioned. Review important changes before sharing or publishing them.

## If the answer misses the mark

- Name the exact resource or section Copilot should use.
- Split a broad request into smaller outcomes.
- Describe what is wrong in the current result instead of repeating the same prompt.
