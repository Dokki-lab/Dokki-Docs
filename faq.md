## People and agents

**What makes Dokki agent-native?**

People and AI agents work from the same workspace context: documents, tables, files, messages, permissions, and published knowledge. Work stays attached to the underlying resources instead of being copied between a chat and separate tools.

**When should I use Copilot versus an agent?**

Use Copilot for an on-demand conversation while you are working. Use an agent for a persistent, scoped teammate that can follow standing instructions, use configured tools, and run scheduled work when appropriate.

## Account and billing

**Is Dokki free?**

Yes. The Free plan includes core editing, limited AI usage, personal storage, and a small number of workspaces.

**When should I upgrade?**

Upgrade when you need more AI credits, stronger models, larger files, more storage, more workspaces, or team controls.

**Can an organization pay for AI usage?**

Yes. Organization workspaces can use shared organization billing and AI balance when configured.

## Publishing

**Do published pages update automatically when I edit the source?**

No. Publishing freezes a snapshot. After editing a published resource, commit changes or publish again to update the public page.

**Can I use my own domain?**

Yes, with custom domain setup. Advanced proxy setups can also support subdirectory publishing when configured correctly.

**Can I hide a site from search engines?**

Yes. Turn off listed mode. The site remains reachable by direct link, but Dokki marks it noindex and removes sitemap.xml and llms.txt for that site.

**Can I remove the Powered by Dokki footer?**

Not currently. Custom domains change the URL, but published pages still show Dokki branding.

## AI and data

**Can Copilot see private resources?**

Only when the current user and tool context have access. Private resources are filtered out for people, agents, and MCP clients without permission.

**Can agents run on a schedule?**

Yes. Agents can run scheduled jobs. Schedules are skipped while a workspace is archived.

**What is Knowledge Graph?**

Knowledge Graph is an experimental organization feature that extracts entities and relationships to improve search and related-entity exploration.

## Files

**Can I upload large files?**

Yes, within your plan and organization limits. Dokki checks quota and file size before signed upload.

**Are files published as docs pages?**

No. Files use resource permissions and links. Publish a document if you want a public page that explains or links to file material.
