# v1.5.0 — Connected Work, Safer Organizations & Faster Search

Released: July 20, 2026 v1.5.0 makes connected work easier to navigate, clarifies Connect and Publish, improves Search and Deep Search, adds Artifact templates, and strengthens organization workspace

Canonical source: [https://dokki.one/pub/docs/v1-5-0-connected-work-safer-organizations-faster-search](https://dokki.one/pub/docs/v1-5-0-connected-work-safer-organizations-faster-search)

**Released:** July 20, 2026

v1.5.0 makes connected work easier to navigate, clarifies Connect and Publish, improves Search and Deep Search, adds Artifact templates, and strengthens organization workspace access.

## What changed

### Connected work

- Documents can show **Referenced by** so you can see where a resource is being used.
- Connect separates **AI Clients**, **MCP Apps**, and **Channels** by purpose.
- AI Clients can show recent activity based on observed use.
- Imports now follow one clear path and link completed items to the resources they created.

### Everyday AI and publishing

- New users can start with the built-in Dokki Agent without installing another agent first.
- Copilot adds clearer conversation filters and message timestamps.
- Publish is organized into Overview, Content, Appearance, and Site settings.

### Search

- Regular search returns more relevant title and keyword matches.
- Deep Search again expands the question, finds sources, and writes a sourced answer.
- Search permission checks are faster when many results are considered.

### Templates and organizations

- Workspace members can find and create independent copies of Artifact templates.
- Organization membership and workspace access are kept separate more consistently.
- Invalid or removed workspace memberships no longer provide access.

## Try it

- Open a referenced document and use **Referenced by** to follow the connection.
- Visit **Connect** and review the AI Clients, MCP Apps, and Channels sections.
- Create an Artifact from a workspace template and confirm that editing the copy does not change the template.
- Ask Deep Search a question that needs several sources.

## Do I need to do anything?

No action is required for normal use. Organization admins should review workspace membership if someone no longer needs access. Existing integrations continue to use their current permissions.
