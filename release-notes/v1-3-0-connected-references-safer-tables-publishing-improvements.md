**Released:** July 19, 2026

v1.3.0 improves how Dokki records relationships between resources, strengthens everyday table and upload behavior, and polishes published content.

## What changed

### Connected references

- Dokki records structured references between resources so related work can be found more reliably.
- Referenced resources can surface where they are being used as the experience becomes available across the product.

### Tables, uploads, and connections

- Table structure changes are more consistent and easier for Dokki tools to handle.
- Upload paths receive stricter checks before a file is accepted.
- External app connection failures show the useful provider error instead of hiding it behind a generic message.

### Published pages

- Article cards with a single cover image use a cleaner and more consistent layout.

## Try it

- Open a document that links to other Dokki resources and confirm the references lead to the intended items.
- Retry a failed external app connection and use the displayed message to correct the setup.
- Review a published article card that uses one cover image.

## Do I need to do anything?

No action is required. Existing documents, tables, uploads, and published pages continue to work.
