Use Publish to create a public site from selected workspace resources. Workspace content stays private until you publish it.

## Open the Publish console

Go to **Workspace → Extensions → Publish**. The console has four areas:

- **Overview** — site address, status, recent publishing activity, and suggested next steps.
- **Content** — resources that can be published, updated, or removed from the site.
- **Appearance** — home-page behavior and tag-based navigation.
- **Site settings** — site address, visibility, languages, and custom domain.

## Publish a resource

- Create or activate the site from **Overview** or **Site settings**.
- Open **Content** and find the document, table, Artifact, file, or folder you want to publish.
- Review the resource for private or unfinished information.
- Choose **Publish** or **Commit** and wait for the status to update.
- Open the public link and check the result as a visitor.

## Understand resource status

- **Unpublished** — the resource is not on the public site.
- **Published** — the public page matches the last version you published.
- **Updated** — the workspace resource changed after the last publish. Choose **Commit** when the changes are ready.
- **Unknown** — Dokki could not confirm the current publishing status. Retry the status check before publishing or removing anything.

## Update or remove public content

Publishing is snapshot-based. Workspace edits do not change the public page until you commit an update. To remove a page from the site, use **Unpublish** and then confirm that its public URL is no longer available.

## Who can publish

Workspace admins and editors can manage publishing. Viewers cannot create a site or publish updates. A resource may also be unavailable when your role on that resource is more limited.

## Troubleshooting

- If the status is **Unknown**, retry before taking another action.
- If a published change is missing, confirm that you committed the updated resource rather than only saving it in the workspace.
- If a public page is visible but missing from navigation, review listing and navigation settings.
