## Vibe a selection

Select text in an editable document and choose **Vibe** from the floating AI controls. Enter an instruction such as make this clearer, explain this, turn this into bullets, or translate it. The selection is packaged as a structured Vibe prompt and sent through Copilot.

Vibe opens the relevant Copilot conversation path; review the returned output before applying it to the document. It is not an invisible background replacement of the selected text.

## What is not an inline command

The current document slash menu does not contain an /ai block or a command that automatically fills a new AI block at the cursor. There is also no cursor auto-completion that you accept with Tab. Use Vibe or the Copilot panel for generation and editing work.

## When to open the panel

Use the AI panel when the task needs attachments, a saved or temporary conversation, resource search, tables or artifacts, external apps, publishing, tool output, or agent handoff. Use Vibe when the task is about one selection in the document you are already editing.

## Permissions and context

Inline AI follows the same permission and model rules as panel chat. It can use permitted local and workspace context, but a selection does not grant access to private resources or connected apps that the acting user cannot use.
