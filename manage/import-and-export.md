Canonical source: [https://dokki.one/pub/docs/import-and-export](https://dokki.one/pub/docs/import-and-export)

Import brings content from supported services into a Dokki workspace. Export creates a copy of a Dokki document in another format.

## Import content

- Open the workspace import flow.
- Choose the source you want to import from.
- Connect or select the external account.
- Choose the content and destination folder.
- Review the scope, then select **Import**.

## Manage import connections

Use **Manage connections** in the import flow, or open **Connect → MCP Apps**, to add, reconnect, or remove the accounts used for imports. Workspace admins manage shared import connections; other members may be able to run an import when their workspace role allows it.

## Follow progress and open the result

The results view shows whether each item is waiting, importing, complete, or failed. Completed items link to the created Dokki resources. Open those links to review titles, formatting, files, and destination folders.

## Supported sources

Available sources can include Google Drive, Notion, Linear, Confluence, SharePoint, OneDrive, DingTalk, Feishu/Lark, and WeCom. The exact sign-in method and content picker depend on the source.

## One-time import and sync

- Use a **one-time import** when you need a current copy in Dokki.
- Use **sync** when the Dokki copy should continue receiving supported updates from the source.
- Sync is one way: editing the Dokki copy does not update the external source.

## Export a document

Open the document's export menu, choose PDF, Markdown, text, Word, or image when available, then download the result. Review the exported file before sending it outside your team.

## Troubleshooting

- If an item fails, open its error details before retrying.
- Reconnect the source account when authorization has expired.
- Confirm that you can read the source item and edit the destination workspace.
- Large imports may take time; use the results view instead of starting the same import again.
