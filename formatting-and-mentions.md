## Inline formatting

Select text to use bold, italic, underline, strike-through, inline code, highlight, text color, and links. Links are structured marks, so changing a visible label does not have to change its destination.

The block selector can switch between paragraph text, Heading 1 through Heading 3, to-do list, bullet list, numbered list, quote, code block, and table. The editor also supports draggable blocks, collapsible headings, character and word counts, and image resize controls.

## Mentions

Type @ in an editable document or comment to open the mention picker. In documents, it can reference workspace members plus documents, tables, and artifacts. These are structured references rather than plain text: resource mentions continue to point to the same resource after it is renamed, and selecting one navigates in the app.

Comment mentions notify the mentioned people and power the comments panel's mentions-me filter. A mention does not override access: opening a referenced private resource still requires permission.

## Formatting boundaries

Inline table cells have their own controls, including cell background colors. Code blocks are syntax highlighted and use a language selector. Use the in-app surfaces for the current keyboard shortcuts rather than relying on generic editor shortcuts from another product.

Copilot can work at the cursor or over selected text through the floating AI controls and tab completion. See **AI Copilot -> Inline AI & Vibe** for the AI workflow.
