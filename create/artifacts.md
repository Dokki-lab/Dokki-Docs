Canonical source: [https://dokki.one/pub/docs/artifacts](https://dokki.one/pub/docs/artifacts)

Artifacts are interactive visual resources. Use them for dashboards, diagrams, calculators, explainers, and other experiences that need more than a document or table.

## Create an Artifact

- Choose **Create → Artifact**.
- Start from a blank Artifact when you want full control, or choose a workspace template when a useful starting point already exists.
- Describe the result you want, including the audience, data, layout, and interactions.
- Review the preview and continue editing until the content and interactions are correct.

## Artifact templates

A template stores a reusable starting version of an Artifact. Creating from a template makes an independent copy, so later changes to the template do not alter Artifacts that already exist.

## Save an Artifact as a template

Open the Artifact settings, choose **Save as template**, then add a name and a short description that tells teammates when to use it. Only save a template after removing one-off or sensitive content.

## Collaborate and publish

Artifacts use the same workspace and resource access controls as other Dokki resources. Publish an Artifact when people outside the workspace should be able to use the approved version.

## Choose the right resource

- Use a **Document** when reading and editing text is the main experience.
- Use a **Table** when rows, columns, filtering, and repeated records are central.
- Use an **Artifact** when visual layout or interaction is part of the answer.
