Press / in an editable document to open the slash menu. Type to filter, use the arrow keys to choose a command, Enter to insert it, and Escape to close the menu.

## Writing and structure

The current menu can insert plain text, to-do lists, inline tables, Heading 1 through Heading 3, bullet lists, numbered lists, quotes, code blocks, and dividers. Inline tables start as a three-by-three grid with a header row.

The node selector in the floating toolbar handles common block conversions. It is hidden in a table cell so cell controls can take priority.

## Media and visual blocks

Use the menu to upload an image from disk, add a YouTube video, embed an X post, or insert SVG markup. SVG is sanitized before it renders.

There are dedicated embed entries for Figma, GitHub, Google Docs/Sheets/Slides, and a generic link embed. The embed block opens an inline URL editor when it has no source. External embeds still depend on the source service's permissions and browser embedding policies.

## Workspace resource embeds

Choose **Embed Resource** to pick a workspace document, table, artifact, file, or folder. The block follows the source resource's current title and type. If the viewer cannot see the source, it shows an unavailable state instead of exposing private content.

## Other commands

**Send Feedback** opens the product feedback page in a new tab. The slash menu does not include an AI command or a math command; use the floating AI controls for writing assistance and the inline-math workflow described in **Embeds & Math**.
