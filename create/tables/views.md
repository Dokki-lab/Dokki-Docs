## Views

- **Table** is the spreadsheet-style editing grid.
- **Kanban** groups rows by a configured column into lanes.
- **Gantt** plots rows on a timeline from a date range or configured start and end date columns.
- **Calendar** places rows on month or week layouts from a configured date column.

Configure Kanban, Gantt, and Calendar from their view settings. If a view looks empty, first check that it has the column type and configuration it needs.

## One shared data source

Changing a cell, row, or column updates the same collaborative table regardless of which layout is open. A form submission also appends a row to that same source.

## Local saved views

The saved-views menu captures the current layout and presentation as a named preset. It includes filters, sort, frozen and hidden columns, widths, wrapping, and the active view configuration.

Saved views are local browser preferences. They are not shared with collaborators and do not create separate table data. Use them to move quickly between your own triage, calendar, or planning layouts; use the shared table data itself for the team source of truth.
